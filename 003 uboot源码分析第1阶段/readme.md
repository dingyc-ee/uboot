# uboot源码分析第1阶段

## 第1节 以前的单片机程序

在讲解uboot源码之前，我们先来回顾一下，以前我们写的硬件部件的实验跟uboot是差不多的。主要有以下的流程：

1. 初始化
    + 观看门狗
    + 初始化时钟
    + 初始化SDRAM
2. 把程序从NAND Flash拷贝到SDRAM
3. 设置SP栈（调用C函数）

所谓的uboot就是个复杂的单片机程序，这些流程当然也得做。uboot做了些什么？

## 第2节 uboot程序

uboot跟裸机程序类似，主要有以下的流程：

1. 硬件相关初始化
2. 设置栈（SP指向内存）
3. 调用C函数，从Flash读出内核，启动内核
